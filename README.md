# 🗂 Task Management System

A web-based Task Management System built using Python and Flask.  
This project helps users manage tasks efficiently with a simple dashboard interface.

---

## 🚀 Features

- ✅ Create, update, and delete tasks  
- 📋 Dashboard view  
- 🗃 Database integration  
- 👤 User-friendly interface  

---

## 🛠 Technologies Used

- Python  
- Flask  
- HTML  
- CSS  
- SQL (Database)

---

## 📂 Project Structure
app.py
templates/
static/
tms_db.sql    

---

## ▶️ How to Run

1. Clone the repository  
2. Open project folder  
3. Install dependencies:
## pip install flask
4. Run the app:
## python app.py
5. Open in browser:
http://127.0.0.1:5000/
